
# RSU-Portfolio

My Personal Portfolio & Project Management System

---

## Overview

RSU-Portfolio is a personal portfolio website designed for a single owner (myself).

The system is divided into two main parts:

- Public Website: Visitors can browse and view my projects
- Admin Dashboard: A private, authenticated system where I can manage my portfolio

This allows me to dynamically update my projects without modifying the frontend code.

---

## Key Features

### Public Features
- View portfolio projects
- Responsive design for all devices

### Admin Features
- Secure login system (admin only)
- Create, update, and delete projects (CRUD)
- Dynamic content management via database

### Contact System
- Visitors can send messages
- Notifications are sent to LINE OA in real-time

---

## Tech Stack

### Frontend
- Next.js
- Tailwind CSS 4

### Backend 
- FastAPI (REST API)

### Database
- PostgreSQL

### Deployment
- Vercel (Frontend)
- Docker (Backend & Database)
